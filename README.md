# 🎵 Melodio – Advanced Web Music Player

Melodio is a modern and interactive **web-based music player** built using **HTML, CSS, and JavaScript**.  
It provides a smooth music listening experience with features like playlist management, search, favorites, recently played tracks, and audio visualization.

---

## 🚀 Live Features

✨ Beautiful and responsive UI  
🎧 Play / Pause / Next / Previous controls  
🔀 Shuffle and 🔁 Repeat modes  
📊 Audio Visualizer  
🔎 Smart Song Search  
❤️ Favorite Songs (Saved in browser)  
🕓 Recently Played Songs  
⏯ Resume Playback after refresh  
📂 Upload your own songs  
⌨ Keyboard Controls Support  
🔊 Volume & Mute Control  

---

## 🖥 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Web Audio API
- LocalStorage
